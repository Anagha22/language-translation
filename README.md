## Instructions on running the application

1. Download the PLTcode directory.
2. Enter the application's root directory.
3. `$ rake db:migrate`
4. `$ rake db:seed`
5. `$ bower install`
6. `$ rails s`
7. Enter `localhost:3000` on a web browser
